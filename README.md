# Welcome to Antd-Admin-Scaffold 👋

## Description
后台脚手架使用 create-react-app 生成 Typescript+React 项目，使用 Antd + UForm + AntV 搭建

## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```
## Generator Test Environment code
```sh
yarn build:test
```

### Generator Prod Environment code
```sh
yarn build:prod
```

## Run tests

```sh
yarn jest
```
